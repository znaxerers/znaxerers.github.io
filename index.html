Hero Kid Misadventures

REPO - ITCH

Game - April 2025

A game made in a week for Indie Jam Student Game Jam 2025.
(Technically, not a student but unemployed graduates could join. (¯ . ¯;) )

A 2D action sidescroller where you play as Hero Kid and have to consume organ fruits to kill Dracula.

I tried to add a lot of juice into the combat loop with stuff like hitstop, hitstun, screenshake & hit effects.

I was the sole programmer & designer and worked with two other artists.

SkeleSkribe DS

REPO

Game - ??? 20XX

A gesture-recognition OCR-based spellcasting mystery dungeon game.

Wanted to incorporate Japanese Kanji OCR gameplay and also mystery dungeon gameplay.

Designing the runes has been really hard because they have to make sense when combined - though coming up with one offs has been pretty easy.

Using the BlocksDS toolchain as they had a gesture recognition example but will probably look into using NFLib and NitroEngine as well so I can finish this faster.

Wondering why am I making a game of a game when the first one doesn't event exist yet?
I don't know I just really liked the idea.
